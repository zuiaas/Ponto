1. Abra o navegador e baixe o **Node JS** em:
   
   ```bash
   nodejs.org/pt
<br>

2. Clone o repositório **Ponto** em uma pasta:
   
   ```bash
   git clone https://github.com/zuiass/Ponto.git
<br>

3. Entre no projeto em:

   ```bash
   cd Ponto
<br>

4. Instale os módulos (node_modules):

   ```bash
   npm install
   ```
<br>

5. Rode o projeto:

   ```bash
   npm run start
<br>
